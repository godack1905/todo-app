# Family Todo App

A very simple self-hosted web application to manage shared tasks for a family or household.

No users, no authentication, no complexity.
Just tasks, persistence, and easy deployment.

## Features

Create tasks

Mark tasks as done

Delete tasks

Persistent storage using SQLite

Single-container Docker setup

CI/CD with GitHub Actions

Easy redeploy with one command

## Tech Stack

- Backend: Node.js + Express
- Database: SQLite (file-based)
- Frontend: Plain HTML + JavaScript
- Containerization: Docker
- CI/CD: GitHub Actions
- Registry: Docker Hub
